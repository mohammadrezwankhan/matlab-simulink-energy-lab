# Contributing

Contributions should make energy-system modeling examples clearer, more reproducible, or easier to validate.

## Good Contributions

- Add small examples.
- Improve setup instructions.
- Add expected results.
- Document assumptions and units.

## Pull Request Checklist

- [ ] Example purpose is clear.
- [ ] Units are stated.
- [ ] Required MATLAB/Simulink products are listed.
- [ ] Results are reproducible from the repository instructions.
